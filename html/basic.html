這裡有一份標準的 HTML 文件

聲明表示文檔類型，並幫助瀏覽器正確顯示網頁。它只能出現一次，在頁面頂部（任何 HTML 標記之前）。HTML 文檔本身以<html>開頭和</html>結尾。<head>包含有關 HTML 頁面相關資訊<body>元素定義了文檔的主體，是所有可見內容的容器，例如標題、段落、圖像、超鏈接、表格、列表等。

  • HTML: 描述網頁的內容和語意結構
  • CSS: 描述網頁的樣式和畫面如何被呈現
  • JavaScript: 是一種腳本(程式語言)，使網頁不只有靜態內容，增加互動

HTML編輯器

HTML檔案文件基本上只是一份純文字檔案，只要是可以編輯純文字的編輯軟體都可以做為HTML的編輯器。將編輯好的文件儲存成副檔名為.html的檔案，就是一份HTML文件。像是內建的記事本就是一種，但是使用程式編輯器可以更有效率的編輯。找到適合自己的編輯器甚至可以讓你的速度提升10倍
怎麼看編輯完的 HTML 網頁實際會長什麼樣子呢？答案很簡單，就是用瀏覽器打開，將 HTML 檔案拖曳到瀏覽器直接打開，或在HTML檔案上按右鍵選擇用瀏覽器打開檔案看看！
