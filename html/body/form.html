HTML 表單是用來讓使用者輸入資料，這些資料可以用來和使用者互動，表單內容填完後可以傳回遠端伺服器，像常見的連絡表單。
<form>標籤中常見會包含 <input>、<textarea>、<select>、 <option>、<button>

<form>的重要屬性

  • action: 用來指定一個位址(URL)，這個位址是告訴瀏覽器當使用者送出表單後，要將表格的內容送去哪邊。
  • method: 用來指定資料傳輸時用的 HTTP 協議
    表單數據可以作為 URL 變量（使用method="get"）或作為 HTTP 後事務（使用method="post"）發送。提交表單數據時的默認 HTTP 方法是 GET。
  • 用來指定瀏覽器要在何處顯示表單送出後伺服器回應的結果。值有這些選項：
    • _self: 顯示在表單所在的當前視窗
    • _blank: 顯示在新頁籤/視窗
    • _parent: 顯示在上一層的視窗 (用在例如如果表單是放在<iframe>中)
    • _top: 顯示在最頂層的視窗
    • framename: 指定在哪個框架中開啟
  • autocomplete: 指示這個表單中的欄位是否啟用瀏覽器自動完成機制。值有這些選項：
    • on: 是，預設值
    • off: 否
